# Cover Theory

## Core Formula

High-click cover = click reason + visual hook + readable hierarchy + mobile proof.

The cover should answer three questions within one glance:

1. Why should I stop?
2. What is this about?
3. Why should I click now?

## Click Reasons

Pick one primary click reason:

- **Pain**: expose a problem the audience already feels. Example: "文字很多怎么排版不乱?"
- **Result**: show a desirable outcome. Example: "3步做出高点击封面"
- **Contradiction**: create tension between expected and actual. Example: "图很好看, 加字反而变丑?"
- **Curiosity gap**: reveal enough to create a question. Example: "为什么这张点击率高3倍?"
- **Avoid loss**: warn about mistakes. Example: "别再把标题塞满了"
- **Identity**: call out a specific audience. Example: "设计新手一定要会"
- **Proof**: show data, before/after, client result, or visible transformation.
- **Conversion**: signal trust, authority, scarcity, offer, or action when the goal is not only clicks.

## Information Hierarchy

Use the three-layer model:

- **Main information**: one short hook. It must be readable in feed preview.
- **Secondary information**: one explanatory line or benefit. It clarifies the hook.
- **Auxiliary information**: labels, numbers, English words, icons, micro-copy. It creates atmosphere and credibility; it does not need to carry the main message.

Do not make every word important. If everything shouts, nothing leads.

## Visual Cognition Rules

- Use contrast to create priority: size, weight, color, brightness, position, blur, and density.
- Use proximity to group related information.
- Use alignment to create trust and reduce mess.
- Use repetition for system feeling: labels, rule lines, badges, numbering.
- Use whitespace as a container for words, not as leftover space.
- Use grid first, then break the grid only where it increases attention.

## Chinese Cover Text Rules

- Break lines by meaning, not by equal length.
- Put the strongest word on its own line when it deserves attention.
- Prefer fewer words for the main hook; move explanation to secondary text.
- Use strong weight for the main hook and stable sans-serif or high-legibility display type for mobile.
- Avoid long vertical Chinese unless the phrase is short and the layout is intentionally poster-like.
- Keep punctuation minimal in the main hook.

## Main Image Semantics

Treat the main image as evidence, not background.

- Face and eyes guide attention.
- Hands and props create direction and specificity.
- Before/after, screenshots, documents, cards, and objects make abstract topics tangible.
- Remove or suppress clutter behind the title.
- If the subject is looking or pointing, place the key message where that direction naturally leads.

## Conversion Layer

When the user wants leads, signups, sales, or trust, add one conversion cue:

- Authority: "10年经验", "企业案例", "实战课"
- Specific result: "点击率+40%", "7天完成"
- Low-friction action: "模板已整理", "可直接套用"
- Scarcity or relevance: "本周直播", "新手版"

Keep the conversion cue secondary. The first job is still to earn the click.
