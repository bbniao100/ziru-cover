# Layout Patterns

Choose the model that matches the script, main image, and click reason. Combine models only when the cover still has one dominant message.

## 1. One-Glance Hook

Use for broad feeds and high-click educational content.

- Put the main hook in the largest type.
- Use one strong color block or object as the attention anchor.
- Let the person look, point, or hold the hook.
- Add a short support line below or beside the main hook.

Best for: "一眼想点", pain, result, curiosity.

## 2. Long-Text Layering

Use when the title or required copy is long.

- Split into main information, secondary information, and auxiliary information.
- Make the main phrase large; move explanations to sidebars, labels, footers, or small cards.
- Treat very small text as atmosphere, not required reading.
- Use boxed regions, color bands, or magazines-style side notes to organize density.

Best for: knowledge covers, course covers, "文字很多怎么排版不乱".

## 3. Grid and Whitespace

Use when images are good but text feels awkward.

- Divide image area, text area, and whitespace before placing details.
- Push images into stable zones and reserve clean space for text.
- Use grid alignment, then add one intentional break for energy.

Best for: clean premium feeling, visual case studies, multi-image covers.

## 4. Magazine Cover

Use when the user wants professional, editorial, or expert positioning.

- Use a clean background or controlled photo.
- Place the person as the issue subject.
- Add title, section labels, English micro-copy, numbers, and footer marks.
- Keep micro-copy low contrast so it adds texture without stealing the hook.

Best for: personal brand, design education, high-trust content.

## 5. Poster / Filmic Split

Use when the topic is emotional, story-driven, or visual.

- Use top area for image/story and bottom area for atmosphere, title, or negative space.
- Use strong spacing and quiet typography.
- Let the cover feel like a complete poster instead of a content card.

Best for: essays, narratives, cinematic topics, brand storytelling.

## 6. Before / After

Use when transformation is the click reason.

- Label "before" and "after" clearly.
- Make the difference visible at thumbnail size.
- Keep explanatory text short; the visual comparison should carry the promise.

Best for: design teaching, tutorials, edits, makeovers.

## 7. Object-as-Hook

Use when the subject can hold or point to a prop, card, screenshot, template, or result.

- Make the object carry the idea.
- Put title on, around, or directionally connected to the object.
- Use the object edge as a layout line.

Best for: practical tutorials, tools, templates, physical demonstrations.

## Platform Defaults

- Xiaohongshu feed: prefer 3:4 or 4:5, large title, strong crop safety.
- Douyin / video account cover: prefer 9:16 or platform-specific crop area; keep title away from UI overlays.
- WeChat article main image: prefer cleaner trust-first hierarchy, less noisy color, strong title clarity.
- Course or training promotion: add proof, result, or offer cue, but keep one primary hook.

## Color and Type Guidance

- Use one attention color, one support color, and one neutral base.
- High saturation is for stopping power; neutral space is for comprehension.
- Avoid letting background and title compete at the same contrast.
- Use larger font size before adding more effects.
- Use stroke, shadow, or backing blocks only to solve readability.
