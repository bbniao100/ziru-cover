# ziru-cover

子儒封面是一个 Codex skill，用于把视频脚本、选题和主图转化成高点击、高转化的中文社交媒体封面方案或完整生图提示词。

一句话说：**输入脚本和主图，输出能用于小红书、抖音、视频号、公众号和课程推广的高点击封面设计方案。**

它适合小红书封面、抖音/视频号封面、公众号主图、课程/训练营推广封面，以及任何需要从“脚本 + 主图”快速产出封面策略、设计提示词或成图的场景。

## 适合谁

- 内容创作者：每周批量做短视频、小红书、视频号封面
- 知识博主：把选题和脚本变成更容易被点开的主图
- 设计师/运营：快速生成封面方向、标题层级和版式说明
- 课程/训练营团队：做更偏转化的活动封面和推广主图
- 没有图像生成工具的用户：直接获得可复制到 ChatGPT Image / image2 的完整提示词

## 能做什么

- 从视频脚本里提炼点击理由
- 生成主标题、副标题、标签和氛围小字
- 选择适合的版式模型
- 输出点击优先、专业优先、转化优先 3 个方向
- 如果当前智能体能调用图像模型，直接生成或编辑封面
- 如果当前智能体不能调用图像模型，输出完整提示词，方便复制到 ChatGPT Image / image2 / 其他官方生图工具

## 工作逻辑

`ziru-cover` 不只是写一句生图提示词，而是按这条链路做判断：

```text
视频脚本/选题
→ 点击理由
→ 主标题/副标题/标签
→ 版式模型
→ 视线路径
→ 封面方向或生图提示词
→ 小图可读性质检
```

默认输出 3 个方向：

- **点击优先**：更强标题、更强冲击，适合拉高打开率
- **专业优先**：更干净、更有信任感，适合个人品牌和专业内容
- **转化优先**：更突出结果、证明和行动暗示，适合课程、咨询和私信转化

## 使用示例

### 示例 1：直接生成封面方向

```text

平台：小红书
目标：提高点击率
视频脚本：很多人做封面时，以为图好看就够了，但真正影响点击的是主图里的视觉路径……
主图：见附件
```

### 示例 2：没有图像模型，只要提示词

```text
用子儒封面。

主图：……
```

```text

```


只提供脚本和主图也可以，skill 会默认按移动端中文信息流封面处理。

## 输出长什么样

如果当前智能体能调用图像工具，skill 会尝试直接生成或编辑封面。

如果当前智能体不能调用图像工具，skill 会输出类似这样的内容：

```text
版本名：点击优先版
点击理由：用强痛点让用户第一眼停下
主标题：封面别再乱排了
副标题：3步做出高点击主图
版式：人物在右，左侧大标题，底部放标签和结果提示
给图像模型的提示词：……
后期加字方案：……
避免项：不要乱码中文、不要改变人物身份、不要让标题太小
```
